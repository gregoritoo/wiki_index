Cette page d’[homonymie](/wiki/Aide:Homonymie "Aide:Homonymie") répertorie les
différents sujets et articles partageant un même nom.

Sur les autres projets Wikimedia :

  * [optimisation](https://fr.wiktionary.org/wiki/optimisation "wikt:optimisation"), sur le Wiktionnaire

Le mot **optimisation** est employé dans plusieurs matières :

  * en [mathématiques](/wiki/Math%C3%A9matiques "Mathématiques"), l' **[optimisation](/wiki/Optimisation_\(math%C3%A9matiques\) "Optimisation \(mathématiques\)")** traite de la recherche d'un extremum d'une fonction, dont les entrées peuvent être soumises à des contraintes ;
  * en [informatique](/wiki/Informatique "Informatique"), l' **[optimisation de code](/wiki/Optimisation_de_code "Optimisation de code")** permet d'améliorer les performances d'un logiciel 
    * plus particulièrement dans les [bases de données](/wiki/Base_de_donn%C3%A9es "Base de données"), l' **[optimisation de requête](/wiki/Optimisation_de_requ%C3%AAte "Optimisation de requête")** définit la meilleure exécution par le [SGBD](/wiki/SGBD "SGBD") d'un code donné,
    * l'[optimisation pour les moteurs de recherche](/wiki/Optimisation_pour_les_moteurs_de_recherche "Optimisation pour les moteurs de recherche") permet d’améliorer le classement d'un site web dans les résultats d'une requête sur les [moteurs de recherche](/wiki/Moteur_de_recherche "Moteur de recherche") ;
  * en [finance](/wiki/Finance "Finance"), l'[optimisation fiscale](/wiki/Cat%C3%A9gorie:Optimisation_fiscale "Catégorie:Optimisation fiscale") permet de minimiser l'imposition légalement.
