# Aide:R

Une page de Wikipédia, l'encyclopédie libre.

Sauter à la navigation Sauter à la recherche

**Wikipédia ne possède pas de[page d'aide](/wiki/Aide:Sommaire
"Aide:Sommaire") avec ce nom.**

* * *

Soit vous avez mal écrit le titre :

  * ![](//upload.wikimedia.org/wikipedia/commons/thumb/6/61/Searchtool.svg/20px-Searchtool.svg.png) [**Recherchez « Aide:R »**](/wiki/Sp%C3%A9cial:Recherche/Aide:R "Spécial:Recherche/Aide:R") dans les autres pages de l'espace Aide.
  * ![](//upload.wikimedia.org/wikipedia/commons/thumb/9/90/Nuvola_apps_khelpcenter.png/20px-Nuvola_apps_khelpcenter.png) Parcourez les **[pages d'aide existantes](/wiki/Aide:Sommaire "Aide:Sommaire")** pour vérifier si la page d'aide que vous cherchez existe sous un autre nom.
  * ![](//upload.wikimedia.org/wikipedia/commons/thumb/5/5d/Sdm.svg/20px-Sdm.svg.png) [Cherchez d'autres pages de Wikipédia pointant vers ce titre](/wiki/Sp%C3%A9cial:Pages_li%C3%A9es/Aide:R "Spécial:Pages liées/Aide:R").

* * *

Soit la page d'aide n'existe pas encore :

  * ![](//upload.wikimedia.org/wikipedia/commons/thumb/e/e2/Nuvola_apps_ksig.png/20px-Nuvola_apps_ksig.png) **[Créez la page](https://fr.wikipedia.org/w/index.php?title=Aide:R&action=edit)**.

* * *

Notes pour les rédacteurs :

  * Si vous avez récemment créé la page et qu’elle apparaît toujours vide, essayez de [purger le cache](https://fr.wikipedia.org/w/index.php?title=Aide:R&action=purge) ou d’attendre quelques minutes avant d’essayer de la recréer.
  * La page a peut-être été supprimée : si tel est le cas, l'historique des effacements est affiché ci-dessus.
  * Elle peut aussi avoir été renommée sans création d'une redirection, jugée inutile.
