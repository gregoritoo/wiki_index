Sur les autres projets Wikimedia :

  * [OP](https://fr.wiktionary.org/wiki/OP "wikt:OP"), sur le Wiktionnaire

**OP** peut désigner :

  * [Observatoire de Paris](/wiki/Observatoire_de_Paris "Observatoire de Paris") ;
  * _Ordo Prædicatorum_ ou [Ordre des Prêcheurs](/wiki/Ordre_des_Pr%C3%AAcheurs "Ordre des Prêcheurs"), couramment appelé Ordre des Dominicains (dont les membres sont désignés par l'abréviation _o.p._ ) ;
  * [Ooty observatory Pulsar](/w/index.php?title=Ooty_observatory_Pulsar&action=edit&redlink=1 "Ooty observatory Pulsar \(page inexistante\)"), désignation pour les [pulsars](/wiki/Pulsar "Pulsar") découverts à l'[observatoire d'Ooty](/w/index.php?title=Observatoire_d%27Ooty&action=edit&redlink=1 "Observatoire d'Ooty \(page inexistante\)") en [Inde](/wiki/Inde "Inde"). Voir [Désignation des pulsars](/wiki/D%C3%A9signation_des_pulsars "Désignation des pulsars") ;
  * [Obscurus Presse](/w/index.php?title=Obscurus_Presse&action=edit&redlink=1 "Obscurus Presse \(page inexistante\)"), un fanzine sur Harry Potter ;
  * [One Piece](/wiki/One_Piece "One Piece"), le manga de Eichiro Oda ;
  * [Opening](/wiki/Opening "Opening"), la musique d'ouverture d'un anime ;
  * [Ouvrier professionnel](/w/index.php?title=Ouvrier_professionnel&action=edit&redlink=1 "Ouvrier professionnel \(page inexistante\)"), par opposition aux ouvriers spécialisés (OS) ;
  * Sur internet, l'opérateur premier ou principal, l'auteur du premier message posté d'un sujet sur les forums de discussion - de l'abréviation anglaise OP pour _Original Post_ ou _Original Poster_ ;
  * [Ordre public](/wiki/Ordre_public "Ordre public") ;
  * [Organisation de producteurs](/wiki/Organisation_de_producteurs "Organisation de producteurs"), ou Organisation paysanne , regroupement de [producteurs](/wiki/Agriculteur "Agriculteur") d'un même produit (lait, légume, fruit, etc.) et souvent d'une même région
  * [Organe producteur](/w/index.php?title=Organe_producteur&action=edit&redlink=1 "Organe producteur \(page inexistante\)") (de molécules aromatiques), dans une [plante aromatique](/wiki/Plante_aromatique "Plante aromatique") ;
  * [Optimum de Pareto](/wiki/Optimum_de_Pareto "Optimum de Pareto") ;
  * [OP Financial Group](/wiki/OP_\(financement_groupe\) "OP \(financement groupe\)"), un groupe coopératif bancaire finlandais basé à Helsinki.
  * [officier de police](/wiki/Officier_de_police "Officier de police")
  * [Officier de protection des réfugiés et apatrides](/wiki/Officier_de_protection_des_r%C3%A9fugi%C3%A9s_et_apatrides "Officier de protection des réfugiés et apatrides")

« **op.** » peut faire référence à :

  * **[Opus](/wiki/Opus "Opus") [![Ce lien renvoie vers une page d'homonymie](//upload.wikimedia.org/wikipedia/commons/thumb/3/3e/Disambig_colour.svg/15px-Disambig_colour.svg.png)](/wiki/Opus "Opus")** : ouvrage, œuvre ;
  * **op. cit.** est l'abréviation de « _Opus citatum_ » qui signifie : ouvrage déjà cité.

**[O.P.](/wiki/O.P. "O.P.")** est le signe de :

  * [Ordre des Prêcheurs](/wiki/Ordre_des_Pr%C3%AAcheurs "Ordre des Prêcheurs") ( _Ordo Prædicatorum_ ), plus connu sous le nom d’ordre dominicain ;
  * Over Powered, terme utilisé dans les jeux vidéo pour designer un élément trop efficace ou compétitif dans l'équilibrage du jeu.
